# HomeLab - Infraestrutura & Redes

Laboratório virtual criado para estudo prático de infraestrutura, redes, sistemas e automação.<br>
O objetivo deste projeto é simular um ambiente corporativo real, permitindo praticar administração de servidores, serviços de rede, monitoramento e troubleshooting.

🎯 Objetivos do Projeto

• Praticar administração Linux e Windows;<br>
• Configurar serviços de rede essenciais;<br>
• Aprender monitoramento de infraestrutura;<br>
• Simular cenários reais de empresas;<br>
• Documentar aprendizados para portfólio profissional.<br>
  Este laboratório é atualizado constantemente conforme novos serviços e tecnologias são estudados.
